# 职教出题-网络营销推广

本技能用于生成中职《网络营销推广》科目的考试题目。

## 题目类型

- **单选题**：4个选项，1个正确答案
- **多选题**：4个选项，2-4个正确答案
- **判断题**：正确/错误判断
- **填空题**：概念名词挖空（遵循特定规则）

## 输出格式

### 单选题
```
题目：[题干]

A. [选项1]
B. [选项2]
C. [选项3]
D. [选项4]

答案：B
解析：[详细解析]
```

### 多选题
```
题目：[题干]

A. [选项1]
B. [选项2]
C. [选项3]
D. [选项4]

答案：ABD
解析：[详细解析]
```

### 判断题
```
题目：[题干]

答案：正确/错误
解析：[详细解析]
```

### 填空题
```
题目：[题干]____是[概念定义]

答案：[概念名称]
```

## 填空题出题规则（核心规则）

### ⚠️ 重要前提：严格对照知识点原文
- **必须从教材/知识点原文出发**，不能自行解释或扩展
- **不能多字少字**：答案必须与原文完全一致
- **不能凭空捏造**：答案必须是教材中明确存在的概念
- **不能改变表述**：保持原文的用词和顺序

### ✅ 正确挖空方式
- **挖空位置**：开头挖空，概念名称放前面
- **示例**：
  - 原文：`网络营销推广活动策略是指商家为实现特定的网络营销目标而制定的整体规划`
  - 挖空：`____是指商家为实现特定的网络营销目标而制定的整体规划`
  - 答案：`网络营销推广活动策略`

### ✅ 适合挖空的内容
- 概念定义（主要形式）
- 核心名词
- 关键术语

### ❌ 禁止挖空的内容
- 记忆口诀（如"准时关全法"）
- 举例内容（如"如双11"、"如直播带货"）
- 数字/细节描述（如"30%"、"第二天"、"1000元"）
- **自行补充的内容**：答案必须是原文已有的概念

### ❌ 常见错误（必须避免）
1. **多字**：如原文是"目标受众"，答案写成"目标受众群体"
2. **少字**：如原文是"电子邮件营销"，答案写成"邮件营销"
3. **换词**：如原文是"营销渠道"，答案写成"推广渠道"
4. **捏造**：答案概念在原文中不存在

## 出题流程

1. **确定知识点**：根据用户指定的章节或知识点确定出题范围
2. **查找概念句**：从教材/知识点原文中寻找"是..."、"是指..."的定义句子
3. **严格对照原文**：
   - 填空题：挖空前半部分
   - 答案必须是原文中的完整概念，不能多字少字
4. **生成题目**：按输出格式生成各类题目
5. **质量检查**：
   - 答案是否与原文完全一致？
   - 是否有自行添加的内容？
   - 概念是否在教材中存在？
