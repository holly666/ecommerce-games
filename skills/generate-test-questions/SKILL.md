# 出测试题技能

## 核心规则

1. **知识点不能超纲，也不能衍生** - 严格按照给定材料出题，不能超出范围或自行延伸
2. **文字表述要多样性** - 每道题的题干表述不能雷同，要变化表达方式
3. **原文挖空出题** - 要和原文一模一样，不能自己编造
4. **格式必须规范** - 严格按照下方示例格式出题

## 题目格式规范

### 单选题
```
1、我国的火警报警电话是（C）？[单选题]    
A、110
B、120
C、119
D、911
解析：我国的火警报警电话是119。
```

### 多选题
```
2、四大文明古国中的，四大文明是下列选项中的哪四个（ABCE）？[多选题]
A、埃及文明
B、印度文明
C、中国文明
D、希腊文明
E、美索不达米亚文明
解析：传统认为，四大文明分别是中国、埃及、印度和两河流域的美索不达米亚。
```

### 判断题
```
3、《出师表》中，"先帝創業未半，而中道崩殂"中的"先帝"是指刘备。（对）
```

### 填空题
```
4、《史记》的作者是？[填空题]
答案：司马迁
```

### 填空题（原文挖空）
```
5、补全以下诗句：{床前明月光}，疑是地上霜。{举头望明月}，低头思故乡。
```

### 简答题
```
6、请简述发生火灾时的操作规范。[简答题]
```

## 输出方式

1. **生成Word文档**：使用python-docx库生成.docx文件
2. **文件名用英文**：避免中文文件名乱码，格式如 `project1_xxx_test_questions.docx`
3. **发送文件**：通过飞书发送Word文档给用户

## 题目数量要求

根据用户需求出题，通常为：
- 单选题：30道
- 多选题：10道
- 判断题：20道
- 填空题：5道
- 简答题：1道

## 注意事项

- 答案解析要简洁明了
- 选项设计要合理（有干扰项）
- 判断题只有对/错两种答案
- 填空题答案要准确
- 简答题答案要点要完整
