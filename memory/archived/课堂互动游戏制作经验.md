- 单文件HTML，无需服务器

### 核心功能
- 开始页面（游戏说明）
- 答题页面（题目+4个选项）
- 结果页面（得分+评语）

### 代码要点
```javascript

        question: "情景描述", 
        options: ["选项A", "选项B", "选项C", "选项D"], 
        answer: 0  // 正确答案索引

// 页面切换：通过style.display控制
// 计分：答对+10分，答错0分

```

## 三、避坑指南

⚠️ **中文引号问题**
- JavaScript中用中文引号（""）会报错
- 解决：统一用英文引号（""）或单引号（''）

⚠️ **飞书传输问题**
- 飞书传HTML文件可能损坏
- 解决：用飞书文档中转，或直接发文件

⚠️ **测试**
- 先用浏览器打开测试通过再发给学生
