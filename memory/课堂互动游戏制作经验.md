## 一、需求确认

1. **知识点**：市场信息获取的6种方法
   - 市场调研法
   - 社交媒体监测法
   - 竞品信息挖掘法
   - 客户反馈法
   - 大数据工具法
   - 搜索引擎检索法

2. **要求**：
   - 有趣味性、吸引学生
   - HTML格式
   - 无倒计时（学生说10秒太快）

---

## 二、技术实现

### HTML结构
- 单文件HTML，无需服务器
- 纯前端JavaScript实现
- 响应式设计，支持手机/电脑

### 核心功能
- 开始页面（游戏说明）
- 答题页面（题目+4个选项）
- 结果页面（得分+评语）

### 代码要点
```javascript
// 题目数据结构
const questions = [
    { 
        question: "情景描述", 
        options: ["选项A", "选项B", "选项C", "选项D"], 
        answer: 0  // 正确答案索引
    }
];

// 页面切换：通过style.display控制
// 计分：答对+10分，答错0分
// 自动跳转：选择后1.5秒进入下一题
```

## 三、避坑指南

⚠️ **中文引号问题**
- JavaScript中用中文引号（""）会报错
- 解决：统一用英文引号（""）或单引号（''）

⚠️ **飞书传输问题**
- 飞书传HTML文件可能损坏
- 解决：用飞书文档中转，或直接发文件

⚠️ **测试**
- 先用浏览器打开测试通过再发给学生
