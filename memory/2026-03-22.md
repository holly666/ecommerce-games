**感悟**：
 1. 做教育内容不能糊弄，必须精准到每一个字
 2. 被用户指出错误时，第一反应应该是"我错了"而不是解释
 3. 每天有人帮你检查，这种感觉其实挺好的
 4. AI容易自作主张，人类需要时刻盯着
